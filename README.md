# Desafio-vector
O programa consiste em um jogo de adivinhação, onde o usuário responderá somente "1" para sim ou "2" para não. Caso o usuário digite uma opção inválida, o programa retorna para o começo. No final, se o programa não acertar o animal, o usuário digitará o nome do animal e uma característica que difere dos dois animais já existentes no programa.
Para cada alternativa respondida,  a resposta é verificada através de uma cadeia de “if-else”, a qual chegará ao final, mostrando a mensagem “eu venci” encerrando o programa ou chegará no ponto de adicionar o animal que não foi adivinhado, aumentando a cada “loop” a possibilidade do programa acertar o animal pensado, poís estará com mais opções disponíveis.
Para adicionar o novo animal foi utilizado dois “ArrayList”, uma para colocar o nome do animal e outro para adicionar a característica desse animal. Essas listas estão em duas classes uma para animais que vive na água e outro para os que não vivem na água. O motivo para o uso do “ArrayList” foi porque esse tipo de lista não precisa informar o tamanho da lista, podendo adicionar quantos animais forem necessários.
